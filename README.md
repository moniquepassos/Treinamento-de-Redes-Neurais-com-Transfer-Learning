# Treinamento-de-Redes-Neurais-com-Transfer-Learning
# Desafio Projeto Final Transfer Learning DIO

## Descrição:
Esse projeto para a plataforma DIO aplica **Transfer Learning** utilizando **ResNet50**, pré-treinada no dataset ImageNet, para classificação de imagens de gatos e cachorros.
O experimento foi desenvolvido em **Python**, utilizando **TensorFlow/Keras**, no **Google Colab**.

## Objetivos:
- Aplicar os conceitos de Deep Learning aprendidos em um cenário real
- Utilizar Transfer Learning com modelos pré-treinados
- Compartilhar o projeto no GitHub

## Modelo Utilizado:
- ResNet50 (pré-treinada no ImageNet)
- Camadas convolucionais congeladas
- Camadas densas adicionadas para classificação binária
  
## Dataset:
- Dataset Cats vs Dogs
- Duas classes: gatos e cachorros
- [Link para Download](https://www.kaggle.com/datasets/tongpython/cat-and-dog)

## Tecnologias Utilizadas:
- Python
- TensorFlow / Keras
- Google Colab
- Matplotlib

## Resultados:
O modelo atingiu boa acurácia já nas primeiras épocas, demonstrando a eficiência do Transfer Learning.

## Conclusão:
O uso da ResNet50 permitiu um treinamento eficiente mesmo com um dataset limitado, evidenciando o poder de modelos pré-treinados em tarefas de visão computacional.
